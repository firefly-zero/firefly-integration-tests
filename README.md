# firefly-integration-tests

Integration tests for [Firefly Zero](https://fireflyzero.com/) using [firefly-test](https://github.com/firefly-zero/firefly-test). These tests ensure that the expected runtime behavior doesn't change between releases.

## Usage

```bash
firefly_cli test
```
